# Empower Sierra Leone - Standalone Header & Hero

This folder contains a standalone version of the navigation header and hero section from the Empower Sierra Leone platform.

## Files

- `index.html` - Main HTML file with navigation and hero section
- `styles.css` - All CSS styles including dark mode
- `script.js` - JavaScript for theme toggle, mobile menu, and scroll effects

## Features

- ✅ Responsive navigation with mobile hamburger menu
- ✅ Dark/Light mode toggle (saves preference to localStorage)
- ✅ Sticky navigation with scroll effect
- ✅ Hero section with background image and gradient overlay
- ✅ Smooth animations and transitions
- ✅ Mobile-first responsive design

## Usage

1. Open `index.html` in your browser
2. The navigation and hero section will be fully functional
3. Toggle between light and dark mode using the sun/moon icon
4. On mobile, click the menu icon to open the navigation menu

## Image Paths

Make sure the following images are accessible:
- `../src/assets/logo-icon.png` - Logo icon
- `../src/assets/hero-bg.jpg` - Hero background image

You may need to adjust the image paths based on your folder structure.

## Browser Support

Works in all modern browsers that support:
- CSS Grid & Flexbox
- CSS Custom Properties (variables)
- localStorage API
- ES6 JavaScript
